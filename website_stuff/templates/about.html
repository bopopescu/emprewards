ABOUT

The purpose of this project was to develop an application that would facilitate the giving and redemption of gift points in a professional setting in the form of a web interface. The users are the UT staff and students and would further be seperated into regular users and admin. The administrators of this application have special actions that only they can perform.

With this service, an admin is able to perform the following actions:

  • Add users: Admins are able to create users that can interact with the system
  • View reports: Admins are able to view aggregate reports on how the system is being used by users
  • Remove users: Admins are able to delete users from the platform
A user can do all of the actions above and more:
  • Give Points: Users can award other users gift points
  • Redeeem Points: Users are able to redeem the points they recieved from others into gift cards
